翻訳

テキスト翻訳

  1. Languageで翻訳方向を指定
    • Target Languageが日本語の場合、「である調」を選択すると、ですます調がである調に変換される
  2. Glossaryで用語集を指定
  3. Text Translatorの左側のテキストエリアに翻訳したい文章を入力
  4. Translate with Google ボタンをクリックまたはCtrl + Enterキーを押下
  5. Text Translatorの右側のテキストエリアに翻訳された結果が表示される

ファイル翻訳

  1. Languageで翻訳方向を指定
    • Target Languageが日本語の場合、「である調」を選択すると、ですます調がである調に変換される
  2. Glossaryで用語集を指定
  3. Choose File ボタンをクリック
  4. 翻訳したいファイルを選択
  5. インライン書式を維持したい場合はDelete Format Tagsのチェックをはずす
  6. Upload ボタンをクリック
  7. アップロードされたファイルの ボタンをクリック
  8. 翻訳がおわるとProgressが Download に変わるのでクリックしてダウンロードする

仕様

  • 英語⇔他言語のみNMTに対応
  • その他の組み合わせはSMT
  • ファイル翻訳は*.pptx、*.docx、*.xlsxのみサポート
  • ユーザの判別はセッションを使用
    • 同じPCでもブラウザを変えると別ユーザ扱い
    • 有効期限は2週間で、なにか操作があるたびに延長
  • 用語集は全ユーザーで共有
  • ファイルは30日で自動削除
  • 管理者画面でIPアドレス、翻訳文字数を保持

用語集

対応ファイルフォーマット

  • カンマ区切り
  • UTF-8
  • 拡張子csv
  • ヘッダー行なし

作成方法

  • Glossaryページを開く
  • Source LanguageとTarget Languageを選択
  • Choose File ボタンをクリック
  • CSVファイルを選択
  • Upload ボタンをクリック
  • アップロードされたファイルの ボタンをクリック
  • アラートが表示され、StatusがCreatedに変わったら作成完了

アップデート予定

  • PDF対応

使い方

テキスト翻訳

  1. Select Languageで翻訳方向を指定
  2. Text Translatorの左側のテキストエリアに翻訳したい文章を入力
  3. Translate ボタンをクリックまたはCtrl + Enterキーを押下
  4. Text Translatorの右側のテキストエリアに翻訳された結果が表示される

ファイル翻訳

  1. Select Languageで翻訳方向を指定
  2. Choose File ボタンをクリック
  3. 翻訳したいファイルを選択
  4. インライン書式を維持したい場合はDelete Format Tagsのチェックをはずす
  5. Upload ボタンをクリック
  6. アップロードされたファイルの ボタンをクリック
  7. 翻訳がおわるとProgressが Download に変わるのでクリックしてダウンロードする

仕様

  • 英語⇔他言語のみNMTに対応
  • その他の組み合わせはSMT
  • ファイル翻訳は*.pptx、*.docx、*.xlsxのみサポート
  • ユーザの判別はセッションを使用
    • 同じPCでもブラウザを変えると別ユーザ扱い
    • 有効期限は2週間で、なにか操作があるたびに延長
  • ファイルは30日で自動削除
  • 管理者画面でIPアドレス、翻訳文字数を保持

アップデート予定

  • Glossary対応
  • PDF対応
